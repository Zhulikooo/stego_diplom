# Автоматизированная система стегоанализа растровых изображений

Автоматизированная система стегоанализа растровых изображений, реализующая RS-метод Фридрих для обнаружения LSB-встраивания информации. Разработана в рамках выпускной квалификационной работы бакалавра.

## 📋 Описание

Система предназначена для:
- Обнаружения скрытой информации в растровых изображениях (PNG, JPEG, BMP) с помощью RS-метода Фридрих
- Встраивания информации в изображения шестью различными LSB-методами для тестирования
- Извлечения скрытого текста из стегоизображений
- Проведения экспериментальных исследований чувствительности RS-метода

## 🚀 Основные возможности

### 1. Стегоанализ
- RS-анализ изображений с расчётом показателей:
  - `R_m` — процент Regular групп
  - `S_m` — процент Singular групп
  - `R_m - S_m` — ключевая разность для обнаружения
  - Заключение: чистое / подозрительное / обнаружена встройка

### 2. Встраивание информации
- 6 методов LSB-встраивания:
  1. Последовательное LSB
  2. LSB в синий канал
  3. Случайные пиксели (LCG)
  4. Случайные биты + случайные пиксели
  5. Только случайные биты
  6. Многобитовое встраивание (1–8 бит в пиксель)

### 3. Извлечение информации
- Восстановление скрытого текста из стегоизображений с маркером начала `0xAABBCCDD`

### 4. Экспериментальные исследования
- Зависимость R-S разности от процента встраивания (0–100%)
- Порог чувствительности (1–20 битов, до 2000 битов)
- Встраивание битов в один пиксель (1–8 битов)
- Встраивание битов во все пиксели (1–8 битов)
